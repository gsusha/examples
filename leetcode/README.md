## LeetCode

В этом разделе собраны решения задач с LeetCode.

## Что есть в каждой задаче

Каждая задача оформлена в отдельной папке и может содержать:
- `task.md` - условие задачи и ключевые ограничения
- `solution.ts` - решение на TypeScript
- `notes.md` - рассуждения, идеи и пояснения к решению
